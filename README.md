# Onda Prodígio — Página de Vendas (VSL)

Página de vendas com vídeo Vturb, em português de Portugal.

## Deploy

Hospedada na [Vercel](https://vercel.com). Cada push para `main` faz deploy automático.

## Vturb

O embed do Vturb vai no `index.html`, dentro de `#vturb-player`. O CTA com delay é gerido pelo próprio Vturb.

## Desenvolvimento local

Abrir `index.html` no browser ou usar um servidor local:

```bash
npx serve .
```
